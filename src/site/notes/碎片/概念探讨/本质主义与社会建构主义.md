# 本质主义
本质主义认为任何的实体（如一只动物，一群人，一个物理对象，一个观念）都有一些必须具备的本质，这些本质是不可变的，永恒的，决定了实体的身份和特征。
举例：男人就是男人，女人就是女人

# 社会建构主义
认知来源于社会建构，人们的自我性别认知并不是由生理性别决定的，而是由社会和文化塑造的。
举例：LGBTQ+
