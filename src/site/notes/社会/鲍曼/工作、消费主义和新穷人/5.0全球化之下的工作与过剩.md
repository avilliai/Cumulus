# 现代化与现代性
现代化不应被解释为“通往现代性之路”，**现代化就是现代性，一旦停止，现代性也随之消亡**。
现代化是对世界和人们生活方式的一种难以抑制、令人上瘾的重塑，是现代生活方式的同义词，永久和持续的现代化是现代性的基本特征。

**现代化被视为一种工具，用于实现理性社会这个终极目标**。
## 资本主导下的现代化逻辑
>资本主导的现代化逻辑违背了⼈们的期望。资本主义初期，其动 ⼒来⾃征服并殖⺠“处⼥地”——把⽣产者和⽣产资料分离，把商业活 动和家庭经济分离，从⽽把⾮现代化的⽣活带⼊市场经济的轨道，并 将其变成资本的牧场

# 解决本土问题的全球化策略之兴衰
**==地球的饱和，本质上意味着过剩⼈⼝处理⾯临严 重危机。在过剩⼈⼝的产⽣有增⽆减，不断上升到新⾼度的同时，地 球上的垃圾场和垃圾回收的⼯具却急剧短缺==**。
## 人类垃圾场的消失
**地球已满负荷。
这是一个政治学和社会学的论断**。
技术进步拓展了人类的活动空间(成本也在增加)，也削弱了很多区域容纳人类居住和耕作的能力，经济进步使过去的谋生方式不再可行、不切实际，进而增加了休耕和废弃的荒地面积。

地球已满负荷指地球上居民的生存方式和方法。它标志着“⽆⼈区”的消 失，标志着能够被界定或被视为⽆⼈居住、没有主权管辖的⼟地的消 失，标志着可以作为殖⺠地的⼟地的消失。这样的⼟地在现代历史的⼤部分时间⾥，在持续的“现代化”进程影响之下，扮演着⼈类垃圾场 的重要作⽤，现在它们基本上不复存在。
## 现代性与废弃的人
**制造无用的人——编外、冗余、不必要、过剩的人口，是现代化过程中不可避免也难以处理的结果，与现代化如影相随**。
>这是秩序建⽴（每种秩序都会丢弃现存⼈⼝中“不协调”“不合适” 或“不受欢迎”的部分）和经济发展（经济发展不可能不破坏和废弃之 前⾏之有效的谋⽣⽅式，也必须剥夺⼀些从业者的⽣计）不可避免的 副作⽤。

**清除和处理全球“现代化”过 程中产⽣的⼈类废弃物，是殖⺠主义和帝国主义征服的最深层意义**。 这两个过程都是因为发展⽔平严重不平衡⼒量差距持续拉⼤的必然结 果，这也导致现代⽣活模式被限制于⼀个相对较⼩的“特权”区域。这 种不平等为现代国家提供了本⼟“⼈⼝过剩”问题的全球化解决⽅案。

但只要现代性(令人沉迷的现代化)仍然是一种“特权”，这种情况就将持续下去。
>当现代化的胜利号⻆响遍世界的每⼀个⻆落，当全体 ⼈类的⽣产和消费都以货币和市场为中介，当商品化、商业化和货币 化进程渗透进⼈类的每个缝隙，本地产⽣问题的全球化解决⽅案也就 不再可⾏，或者说本地过剩⼈⼝的全球出⼝⽅案不再能够实施。所有 地⽅，尤其是⾼度现代化的地⽅，现在都要承受现代性取得全⾯胜利的后果。他们必须寻找这个全球性问题的本地化解决⽅案，但他们逐 渐认识到，这种努⼒是徒劳的。

**现代生活方式在全球范围内的广泛传播正使得越来越多的人失去了以前的生存方式，以及生物学和社会、文化意义上的生存手段**。
**由此产生的人口压力，即不能“回收”，又很难“安全处理”**。全球人口过剩造成的就业难题和福利难题已经成为严重问题，“移民”和“难民”成为当代政治议程新的核心，模糊而又无所不在的“安全担忧”在新兴的全球战略和权力斗争中发挥着越来越重要的作用。
>在新兴的全球战略和权⼒⽃争中发挥着越来越重要的作⽤。 迄今为⽌，基本上不受管制的、政治不可控的全球化进程，在全 球“流动空间”中建⽴起各种新的“前沿阵地”，曾经仅限于现代主权国 家内部的许多⾏政权⼒被转移到这⾥。众所周知，这些“政治⾃由区 域”的脆弱⼜危险的平衡，是建⽴于“互相保证的脆弱性”之上。

公众的关注和个体的焦虑从问题的经济、社会根源转移到个人人身安全。蓬勃发展的“安全产业”迅速成为废弃物产业的主要分支之一，并成为解决废弃物处理问题的首要手段。
